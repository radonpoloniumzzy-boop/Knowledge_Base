# Keep classification non-blocking and preserve human ownership

自动分类、标签和衍生产物属于非阻塞知识增强。资料完成核心处理后即可成为当前版本；分类未完成时进入未归类，暂不自动加入依赖标签的能力包。自动结果必须携带置信度、来源和证据，人工添加、修改或确认的标签优先且不得被后续自动分类覆盖。

## Considered Options

- 等待自动分类后才开放资料：能力包状态完整，但分类故障会阻塞知识使用。
- 自动结果始终覆盖现有标签：保持模型最新，但会反复推翻用户纠正。
- 分类独立执行且人工判断优先：采用此方案，以保持资料可用性和反馈有效性。

## Consequences

- 未归类资料可检索，但不自动进入依赖标签的能力包。
- 标签关系必须区分自动来源与人工来源。
- 重新分类只能更新非人工拥有的结果。
