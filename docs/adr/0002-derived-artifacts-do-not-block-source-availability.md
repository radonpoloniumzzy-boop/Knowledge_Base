# Do not let derived artifacts block source availability

资料版本完成核心处理后即可成为当前版本。Structure、SOP 和 Insight 作为独立衍生产物生成、重试和更新，其失败不阻塞标准知识文档的检索与使用；依赖特定衍生产物的能力包必须显式报告缺失项，不能静默忽略。

## Considered Options

- 等待全部衍生产物成功后再开放资料：状态统一，但任一 AI 请求失败都会阻塞可用知识。
- 核心处理完成后开放资料，衍生产物独立推进：采用此方案，以换取更高可用性和独立重试能力。

## Consequences

- 工作进程需要分别显示“资料可用”和各类衍生产物状态。
- 更换模型或提示词只需重新生成相关衍生产物，不需要重新导入资料。
