# 06 — 扩展文档提取和质量阻断

**What to build:** PDF、Word、PowerPoint、Excel 等现有支持格式通过可替换的提取 adapter 进入相同核心处理路径，并以可解释规则区分质量阻断与质量警告。

**Blocked by:** 03 — 完成可原子恢复的文本核心处理；04 — 加入失败分类、自动重试和暂停继续

**Status:** done

- [x] 现有声明支持的文档格式都通过统一提取 seam 进入核心处理
- [x] 测试 adapter 可以稳定模拟成功、临时失败、确定性失败和损坏输出
- [x] 不支持、损坏、加密、空提取和明显错误页进入质量阻断
- [x] 无法验证标准知识文档写入或分块完整性时进入质量阻断
- [x] 短内容、低文字量和普通内容疑点只产生质量警告
- [x] 质量阻断说明原因和可采取的修正动作
- [x] 文档提取失败不会留下正式标准知识文档或分块
