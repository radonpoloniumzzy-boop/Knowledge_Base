# 07 — 拆出非阻塞知识增强

**What to build:** 自动分类、标签、Structure、SOP 和 Insight 在资料可用后独立执行和重试，任何单项失败都不影响标准知识文档。

**Blocked by:** 04 — 加入失败分类、自动重试和暂停继续；05 — 加入内容指纹和资料版本

**Status:** ready-for-agent

- [ ] 当前版本可用后才启动知识增强任务
- [ ] 分类、Structure、SOP 和 Insight 拥有独立状态和失败原因
- [ ] 任一知识增强失败不改变当前版本可用性
- [ ] 未完成分类的知识资料显示未归类且仍可检索
- [ ] 未归类资料不自动进入依赖标签的能力包
- [ ] 自动标签保存置信度、来源和证据
- [ ] 自动重新分类不会覆盖人工标签
- [ ] 不适合生成 SOP 的资料可以记录不适用，而不是失败
- [ ] 更换提示词或模型只重新生成目标衍生产物
