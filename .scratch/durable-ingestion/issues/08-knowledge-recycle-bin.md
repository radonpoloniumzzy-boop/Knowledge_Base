# 08 — 加入知识资料回收站

**What to build:** 用户删除知识资料时，整份受管理知识进入可恢复的三十天回收站，并从所有正常使用路径中隔离。

**Blocked by:** 05 — 加入内容指纹和资料版本

**Status:** ready-for-agent

- [ ] 删除知识资料会隔离其资料版本、标准知识文档、分块、标签关系和衍生产物
- [ ] 回收站内容不出现在检索、资料库、统计或新能力包装配中
- [ ] 三十天内恢复知识资料会恢复受管理关系和原当前版本
- [ ] 永久清理只删除系统受管理内容，不触碰无关个人文件
- [ ] 正在处理的知识资料先安全停止，再进入回收站
- [ ] 已导出的能力包快照不因源资料删除、恢复或永久清理而改变
- [ ] 删除旧版本与删除整份知识资料是明确不同的操作
