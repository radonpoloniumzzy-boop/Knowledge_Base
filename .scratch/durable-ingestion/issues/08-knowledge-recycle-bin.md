# 08 — 加入知识资料回收站

**What to build:** 用户删除知识资料时，整份受管理知识进入可恢复的三十天回收站，并从所有正常使用路径中隔离。

**Blocked by:** 05 — 加入内容指纹和资料版本

**Status:** done

- [x] 删除知识资料会隔离其资料版本、标准知识文档、分块、标签关系和衍生产物
- [x] 回收站内容不出现在检索、资料库、统计或新能力包装配中
- [x] 三十天内恢复知识资料会恢复受管理关系和原当前版本
- [x] 永久清理只删除系统受管理内容，不触碰无关个人文件
- [x] 正在处理的知识资料先安全停止，再进入回收站
- [x] 已导出的能力包快照不因源资料删除、恢复或永久清理而改变
- [x] 删除旧版本与删除整份知识资料是明确不同的操作

## Delivery

- schema version 8 已在正式数据库副本演练后应用，升级前后关键记录数量一致。
- 全量自动化测试 56 项通过，主要页面与新增回收站页面均返回 HTTP 200。
- 回收期间知识增强安全暂停，恢复后重新排队；过期清理由后台定期执行。
