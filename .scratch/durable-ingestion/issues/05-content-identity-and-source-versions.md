# 05 — 加入内容指纹和资料版本

**What to build:** 系统以内容而非文件名识别重复资料，并通过不可变资料版本安全处理更新，使失败的新版本不影响已有知识。

**Blocked by:** 03 — 完成可原子恢复的文本核心处理

**Status:** done

- [x] 每个上传内容产生稳定内容指纹
- [x] 完全相同的内容不会创建重复资料版本或重复导入任务
- [x] 重复上传结果提供现有知识资料入口
- [x] 同名但内容变化的资料创建新的不可变资料版本
- [x] 新版本处理期间旧的当前版本继续参与检索和能力包装配
- [x] 新版本只有在核心处理成功后才切换为当前版本
- [x] 新版本失败或进入需要处理时不改变当前版本
- [x] 资料详情可以辨认当前版本和历史版本
